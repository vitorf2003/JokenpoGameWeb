Seja Bem-Vindo(a)

Todas as partidas são realizadas contra a máquina e o jogador
assim que clicar no botão poderá jogar 3 vezes seguidas e depois
que finalizar poderá clicar no botão novamente e recomeçar a partida.

Cada jogada e mensagem de vitória/derrota/empate é feita através de uma
caixa de popup do navegador.

Utilize os seguintes comandos (números) para realizar as jogadas:
1 = Pedra
2 = Papel
3 = Tesoura


Este jogo é simples e não conta com um sistema de placar.
